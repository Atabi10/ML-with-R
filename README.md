# ML with R

Machine Learning project with R focusing on the modelling process.

## Contents

- `modelling process.r` - R script demonstrating the machine learning modelling process

## Getting Started

### Prerequisites

- R 4.0 or higher
- Required packages: `rsample`, `modeldata`

### Installation

Install required packages:

```r
install.packages(c("rsample", "modeldata"))
```

## Usage

Run the modelling process script:

```r
source("modelling process.r")
```
